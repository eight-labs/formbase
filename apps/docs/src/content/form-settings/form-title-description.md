# Form Title & Description

Clear titles and descriptions make it easy to scan submissions and email alerts.

## Example 1: Update the title

1. Open your form settings.
2. Edit the **Title** field.
3. Save changes.


## Example 2: Add a descriptive summary

Add a description that explains where the form lives or who it targets.

- **Title:** Contact Form
- **Description:** Messages from the marketing site footer

## Tips

- Use client names if you manage multiple sites.
- Keep titles consistent with your site navigation.
